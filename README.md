# 单链表通讯录
通过单链表储存一套包含姓名、电话、email、备注字段的通讯录。
## 项目结构
```text
└───LinkListConcact 
    ├── contact.h   // 定义结构体和函数声明
    ├── function.c  // 实现链表增删查改等功能的函数
    ├── init.c      // 实现链表初始化的相关函数
    └── main.c      // 主程序入口，调用各模块实现功能
```
